# CWKA Engineering Wiki

This repository contains the Engineering Department wiki, configured for GitHub Pages.

## Structure

- `_pages/` - Organized wiki content by category
  - `overview/` - Role descriptions and project delivery overview
  - `workflows/` - Step-by-step workflows for FE and PE
  - `standards/` - Drafting standards, layer organization, folder structure
  - `tools/` - Epicor, time entry, and approvals documentation
- `_config.yml` - Jekyll configuration for GitHub Pages
- `index.md` - Main landing page

## Viewing the Wiki

The wiki is published via GitHub Pages. Visit the site at: `https://cwka-engineering.github.io`

## Contributing

To update documentation:
1. Edit the appropriate markdown file in `_pages/`
2. Commit and push changes
3. GitHub Pages will automatically rebuild the site
