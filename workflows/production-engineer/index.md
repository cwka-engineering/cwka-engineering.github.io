# Production Engineer (PE) Workflow

The Production Engineer (PE) workflow is focused on preparing jobs for production. This workflow begins after receiving a job release from the Fabrication Engineer (FE).

> **Related Documents**: [Project Delivery Overview](/overview/project-delivery.html) | [FE to PE Release SOP](/workflows/fe-to-pe-release.html) | [Layer Organization](/standards/layer-organization.html)

## Workflow Stages

1.  **[Setup & Organization](/workflows/production-engineer/setup.html)**
    - Templates, file structure, and layer management.
2.  **[Model Processing](/workflows/production-engineer/processing.html)**
    - Grasshopper scripts, nesting, and quality checks.
3.  **[Deliverables](/workflows/production-engineer/deliverables.html)**
    - Master Parts List, Lay-Up Summary, and CAM output.
4.  **[Troubleshooting](/workflows/production-engineer/troubleshooting.html)**
    - Common issues and solutions.
