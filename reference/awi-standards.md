# AWI Standards Reference

Internal reference for AWI Premium and Custom grade requirements. All CWK/DFW products are produced to **AWI Custom** or **AWI Premium** grade.

> **Default grade**: If no grade is specified in contract documents, **Custom grade applies** (ANSI/AWI 0620 §3.4c).

*Source: AWI 100-2018, AWI 200-2018, AWI 300-2018, ANSI/AWI 0620-2018. For contractual compliance always verify against the full published standards.*

> See also: [ANSI/HPVA HP-1 Reference](/reference/ansi-standards.html) — veneer grading, matching types, species rules, and core types

---

## Submittals (AWI 100)

### Cover Sheet Requirements

Every submittal package requires a cover/title sheet showing:

- Project name and address
- **Specified AWI grade**
- Scope description
- Aesthetic and performance grade
- Duty level
- Contact information for: design professional, GC, manufacturer, installer, and finisher

### Drawing Scale Minimums

| View | Scale |
|---|---|
| Reference Plan | ¼" = 1'-0" |
| Plan & Elevations | ⅜" = 1'-0" |
| Sections | 1½" = 1'-0" |
| Details | 3" = 1'-0" |

Sheet size: minimum 11×17, maximum 24×36. Sequential numbering and dating required on all sheets.

### Casework Drawings Must Include

- Trim/scribe at full-size profile; plan, elevation, and section views
- Dimensions for cabinet construction and face frame attachment
- Drawer member type and thickness (heights and depths), door type and thickness, section of each cabinet type
- All joinery and connection details
- Cabinet attachment to walls and floors
- Blocking/strapping requirements with dimensions A.F.F.
- Provision for field dimensions

### Countertop Drawings Must Include

- Plan view with field joints, joint fastener type/quantity/layout
- Sink size, type, mounting method, and location
- Support brackets (if furnished by others)
- Attachment to casework (fastener type, quantity, layout)
- Wall-mounted tops: build-up and support locations, attachment method
- Elevation view with front/end overhang, edge types, splash type/height, drip groove, end splash return

### Material List Requirements

- Panel core type and thickness — note any moisture-resistant, fire-retardant, NAF, ULEF, or CARB compliance
- Laminate ISO 4586-3 grade and thickness
- All adhesive types with locations and detail drawing callouts
- Hardware with documented specifications
- Veneer: species, cut, leaf match/balance, panel match, room match, flitch number and supplier (if specified)

### Finish Samples

| Sample Type | Minimum Size | Sets Required |
|---|---|---|
| Panel samples | 8×8" [203×203 mm], ¼" [6.4 mm] thick | — |
| Lumber samples | Species-width practical, ¾" [19.1 mm] thick × 8" [203 mm] long | — |
| Transparent / stained | — | 3 sets of 3 samples each, showing range of color and grain |
| Opaque | — | 3 samples per color |

### Review Timeline

The design professional must return one marked copy within **10 business days** of submittal receipt.

## Materials (AWI 300)

### Milled Solid Wood Thickness

Machine sanding permits an additional ¹⁄₃₂" [.8 mm] reduction from milled dimensions.

| Nominal | Rough | Milled |
|---|---|---|
| 4/4 | 1" [25.4 mm] | ¾" [19.1 mm] |
| 5/4 | 1¼" [31.8 mm] | 1" [25.4 mm] |
| 6/4 | 1½" [38.1 mm] | 1¼" [31.8 mm] |
| 8/4 | 2" [50.8 mm] | 1¹¹⁄₁₆" [44.4 mm] |
| 10/4 | 2½" [63.5 mm] | 2³⁄₁₆" [55.6 mm] |
| 12/4 | 3" [76.2 mm] | 2¹¹⁄₁₆" [68.2 mm] |
| 16/4 | 4" [101.6 mm] | 3¹¹⁄₁₆" [93.6 mm] |

### Hardwood Glue-Up Thresholds

**Gluing for thickness**: permitted when finished dimension exceeds **1¹⁄₁₆"** [27 mm].

**Gluing for width**: permitted when finished dimension exceeds **5½"** [139.7 mm] — or **4¼"** [108 mm] for alder, black cherry, select white/red birch, maple, quarter/rift-sawn white/red oak, black walnut, and white ash.

**Glue-up color and grain matching at transparent finish:**

| Premium | Custom |
|---|---|
| Well matched for color and grain | Compatible for color and grain |

### Hardwood Characteristics — Transparent Finish

| Characteristic | Premium | Custom |
|---|---|---|
| Max characteristics per board | 2 (no knots/pitch within 48" of each other) | 3 (no knots/pitch within 36" of each other) |
| Bark pocket | None | None |
| Burl, sound | ≤ ½" dia. | ≤ 1" dia. |
| Check | ≤ ¹⁄₃₂" W × 4" L | ≤ ¹⁄₁₆" W × 6" L |
| Heartwood (select white ash, birch, maple) | None | None |
| Honeycomb | None | None |
| Knot, sound & tight | ≤ ⅛" dia. | ≤ ¼" dia. |
| Knot, checked | None | ≤ ¼" dia. |
| Knot hole | None | None |
| Mineral stain | None | 10% of face |
| Sapwood (cherry, walnut) | ≤ 5% of face | ≤ 10% of face |
| Shake | None | None |
| Split | ≤ ¹⁄₃₂" W × 4" L | ≤ ¹⁄₁₆" W × 6" L |
| Wormhole, filled | None | ≤ ¹⁄₁₆" dia. |

### Hardwood Characteristics — Opaque Finish

| Characteristic | Premium | Custom |
|---|---|---|
| Min face area before any characteristic | 400 in² | 300 in² |
| Additional area per characteristic | 200 in² | 150 in² |
| Max per board | 3 | 4 |
| Min spacing (knots/pitch) | 36" | 24" |
| Knot, sound & tight | < ⅜" dia. | < ⅝" dia. |
| Knot hole, filled | < ⅛" dia. | < ¼" dia. |

### Panel Product Rules

- Odd number of plies, balanced construction, rigid glue line — no delamination
- **Veneer core is not permitted for cabinet door or drawer front components**
- Telegraphing: shall not exceed .004" [.1 mm] in any 3" [76.2 mm] span
- No lamination over existing laminate
- Panel grain direction indicated by size listing — e.g., 48×96 means grain runs with the 96" dimension

## Installation Tolerances (AWI 0620)

### "Inconspicuous" Inspection Distance

| Premium | Custom |
|---|---|
| Not readily visible at **24"** [610 mm] | Not readily visible at **48"** [1219 mm] |

### Gap Tolerances at Field Joints

| Joint Type | Premium | Custom |
|---|---|---|
| Wood to wood (flat & shaped) | .016" [.4 mm] | .031" [.8 mm] |
| Wood to non-wood (flat & shaped) | .031" [.8 mm] | .063" [1.6 mm] |
| Non-wood to non-wood (flat) | .031" [.8 mm] | .063" [1.6 mm] |
| Non-wood to non-wood (shaped) | .063" [1.6 mm] | .094" [2.4 mm] |

Gaps within tolerance may be filled with color-compatible material.

### Flushness at Field Joints

| Joint Type | Premium | Custom |
|---|---|---|
| Wood to wood (flat) | .016" [.4 mm] | .031" [.8 mm] |
| Wood to wood (shaped) | .031" [.8 mm] | .047" [1.2 mm] |
| Wood to non-wood (flat & shaped) | .031" [.8 mm] | .047" [1.2 mm] |
| Non-wood to non-wood (flat) | .031" [.8 mm] | .047" [1.2 mm] |
| Non-wood to non-wood (shaped) | .047" [1.2 mm] | .063" [1.6 mm] |

### Reveal Overlay Tolerances — Frameless Casework

**Reveal overlay ("X" = ⅛" [3.2 mm] nominal):**

| Premium | Custom |
|---|---|
| ±.031" [±.8 mm] | ±.063" [±1.6 mm] |

**Flush overlay ("X" ≤ ⅛" [3.2 mm], "Y" ≤ ¹⁄₁₆" [1.6 mm]):**

| Premium | Custom |
|---|---|
| ±.031" [±.8 mm] | ±.063" [±1.6 mm] |

### Cabinet Door Flatness & Warp

| Premium | Custom | Max any door |
|---|---|---|
| .031" per 12" [.8 mm per 305 mm] | .047" per 12" [1.2 mm per 305 mm] | ¼" [6.4 mm] |

### Scribes and Fillers

| Requirement | Premium | Custom |
|---|---|---|
| Scribe moulding | **Not permitted** | Permitted |
| Max scribe filler width | 2" [50.8 mm] | 3" [76.2 mm] |
| Color-compatible caulk | Permitted (≤ ⅛" [3.2 mm]) | Permitted (≤ ⅛" [3.2 mm]) |

Fastener cover caps on exposed interior surfaces: required at both grades. Exposed fasteners on exterior surfaces (except access panels): not permitted at either grade.

## Premium vs. Custom — Key Differences

The most common areas where Premium imposes noticeably tighter constraints than Custom:

| Topic | Premium | Custom |
|---|---|---|
| Scribe moulding | Not permitted — fillers only, max 2" wide | Permitted, max 3" wide |
| "Inconspicuous" inspection distance | 24" | 48" |
| Gap / flushness tolerances | Roughly half of Custom across the board | Baseline |
| Transparent finish matching | Well matched for color and grain | Compatible for color and grain |
| Mineral stain (transparent hardwood) | Not permitted | 10% of face |
| Veneer end-match alignment | ³⁄₁₆" [5 mm] | ⅜" [10 mm] |
| Cabinet door warp | .031" per 12" | .047" per 12" |
| Wall panel warp | .028" per 12" [.7 mm] | .039" per 12" [1.0 mm] |
