# Drawer Construction

Applies to drawers in base cabinets (see [Base Cabinets — Stretcher Count by Type](/reference/construction-standards/casework/base-cabinets.html#stretcher-count-by-type) for stretcher placement around drawer openings) and to pencil drawers.

## Drawer Front

| Parameter | Value |
|---|---|
| Standard height (top drawer) | 6" unless otherwise noted |
| Inset from cabinet exterior sides | 1/16" |
| Reveal from adjacent doors | 1/8" |
| Reveal at top of cabinet | 1/4" |

## Drawer Box

| Parameter | Formula |
|---|---|
| Box width (side-mount slides) | Opening width − 2 × (slide thickness + 0.03") |
| Box width (undermount slides) | Opening width − 8mm |
| Box depth | Cabinet depth − 2-1/2", rounded **down** to next available drawer slide length |
| Box height | Drawer front height − 1-1/2", rounded **down** to next 1/2" increment |
| Material (sides/back/subfront) — shop-built | 1/2" MDF MEL02 |
| Material (sides/back/subfront) — buyout | 5/8" Maple |
| Material (bottom) — shop-built | 1/4" MDF MEL02 |
| Material (bottom) — buyout | 1/4" Maple veneer |

> **Note:** Two construction paths exist side by side — a shop-built MDF MEL02 box and a buyout Maple box. Confirm which applies per job/spec before pulling dimensions.

## Drawer Slides

| Parameter | Value |
|---|---|
| Type (standard) | Undermount |
| Type (alternate) | Side-mount — Accuride 3832 ESC |
| Mounting position (vertical, side-mount) | Centered on drawer box height |
| Mounting position (horizontal) | Flush with front of cabinet side |
| Available lengths (undermount) | Standard increments (12", 15", 18", etc.) |
| Available lengths (side-mount) | Standard increments (18", 20", 24", etc.) |

> **Note:** Undermount is the standard slide type; side-mount (Accuride 3832 ESC) is the alternate, used only when specified. The vertical mounting-position rule above is specific to side-mount slides — undermount hardware has its own manufacturer-defined mounting position.

## Drawer Pull Placement

| Parameter | Value |
|---|---|
| Horizontal | Centered side to side |
| Vertical | 1-1/2" down from top edge of drawer front (to center) |

## BDS3 Graduated Drawer Heights

| Drawer | Front Height | Box Height |
|---|---|---|
| Top (DB1) | 6" | 4.0" |
| Middle (DB2) | ~8.5" | 7.0" |
| Bottom (DB3) | ~14" | 12.5" |

## BDS4 Drawer Heights

Top drawer is 6" standard. The remaining three drawers are **equally spaced** in the remaining opening height.

---

## Pencil Drawers (PD1D, PD2D)

Pencil drawers are shallow drawer units, typically installed in the knee space of desks or workstations.

- Same drawer box construction rules as standard drawers, above
- Standard drawer front height = **5.25"** (not the standard base-cabinet drawer height of 6" — a common source of confusion given they otherwise share drawer box rules)
- Side-mount slides flush with front of cabinet side
- Pull centered side to side, 1-1/2" down from top edge
