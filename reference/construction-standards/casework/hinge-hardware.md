# Hinge Hardware & Cup Boring

This page covers the concealed-hinge hardware system itself — cup boring, mounting-plate selection, and the resulting door overlay. It applies the same way regardless of cabinet family; for **vertical hinge position** (how far from the top/bottom of the door a hinge sits), see the Hinge Placement section on [Base Cabinets](/reference/construction-standards/casework/base-cabinets.html#hinge-placement), [Upper Cabinets](/reference/construction-standards/casework/upper-cabinets.html#hinge-placement), or [Tall Cabinets](/reference/construction-standards/casework/tall-cabinets.html#hinge-placement).

> This page stays at the system level intentionally. The actual boring-distance/plate-height numbers that produce a given overlay are specific to each hinge and mounting-plate model — **always confirm them against the manufacturer's documentation for the exact SKU in use** rather than assuming a number from a different hinge carries over.

## Standard vocabulary

Blum's own catalogs and spec sheets use a consistent set of abbreviations — worth knowing before reading any hinge spec sheet:

| Abbreviation | Meaning |
|---|---|
| B | Boring distance |
| H | Plate height |
| OL | Door overlay |
| P | Door protrusion |
| R | Reveal |
| S | Side arm protrusion |
| T | Door thickness |
| W | Side panel width |
| FD | Fixed distance |

## How cup position and overlay are determined

The hinge cup bore is a standard 35mm diameter, with mounting screw holes at a fixed 45mm spacing (Ø8) — the same 32mm-system logic already used for [shelf-pin line boring](/reference/construction-standards/casework/carcase-and-line-bore.html#32mm-system).

The cup's position relative to the door edge is set by the **boring distance (B)** — a small standardized index (not a mm value the engineer picks freely). **B**, combined with the **mounting plate height (H)**, determines the resulting **door overlay (OL)**: different B/H combinations produce different overlays on the same hinge.

This is the actual engineering decision behind "selecting the right hinge cup and offset plate": choosing a B/H combination that produces the overlay the design calls for. The exact numeric table mapping B and H to a resulting OL is defined per hinge/mounting-plate model in the manufacturer's documentation — it is not a single universal table across all hinges.

## Not covered here

Physical post-installation adjustment (side/height/depth adjustment screws on an already-mounted hinge, used to fine-tune a hung door within a small tolerance range) is installer-level work, not an engineering specification concern, and is intentionally not documented on this page.
