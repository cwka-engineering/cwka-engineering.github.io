# Tall Cabinets

Tall cabinets follow the universal [Carcase Assembly & 32mm System](/reference/construction-standards/casework/carcase-and-line-bore.html) rules, with a fixed-shelf requirement and dual line-bore zones unique to this cabinet family.

## Standard Dimensions

| Parameter | Value |
|---|---|
| Standard height | 80" |
| Standard depth | 25" |
| Side thickness | 3/4" |
| Nailer thickness | 0.75" (shop plywood — same as base and upper) |

## Fixed Shelf Requirement

**Cabinets over 72" tall must have a fixed shelf at approximately mid-height.** This divides the interior into upper and lower compartments, each with its own independent line bore zone — see [Carcase Assembly & 32mm System — Tall Cabinet Special Case](/reference/construction-standards/casework/carcase-and-line-bore.html#tall-cabinet-special-case).

## Hinge Placement

| Parameter | Value |
|---|---|
| Top hinge from top of cabinet | 3" |
| Bottom hinge from bottom of cabinet | 3" |
| Intermediate hinges | Equally spaced between top and bottom hinges |
| Sixth hinge | Add if door > 90" high |
| Hinge vs. fixed shelf | Adjust hinge location to maintain clearance from fixed shelf if needed |

> This covers vertical hinge position only. For cup boring, mounting-plate selection, and overlay, see [Hinge Hardware & Cup Boring](/reference/construction-standards/casework/hinge-hardware.html).

## Door Alignment

| Parameter | Value |
|---|---|
| Inset from exterior sides | 1/16" |
| Top alignment | FLUSH with top of cabinet (not 1/4" reveal like base) |
| Bottom alignment | FLUSH with bottom of cabinet |

## Pull Placement

| Parameter | Value |
|---|---|
| Inset from side of door (to center) | 1-1/2" |
| Height | 40" up from finished floor (to center), unless otherwise noted |

## Internal Layout
{: #internal-layout}

1. **Top nailer** — 4" × 0.75", pocket screwed at top
2. **Upper adjustable shelf zone** — line bore from 6" below top to 6" above fixed shelf
3. **Fixed shelf** — at approximately mid-height, doweled
4. **Nailer pairs** — flanking the fixed shelf (above and below)
5. **Lower adjustable shelf zone** — line bore from 6" above bottom to 6" below fixed shelf
6. **Bottom nailer** — 4" × 0.75", 3" above bottom
7. **Back panel** — 1/4", dadoed, same rules as base cabinets
