# Casework Type Codes & Names

Full naming reference for standard cabinet, filler, soffit, die wall, and panel types. Use this page to look up what a code stands for, or cross-reference from a construction page back to the full type list.

> See also: [Part Naming](/workflows/fabrication-engineer/part-naming.html) for how these type codes fit into full Epicor part numbers.

| Name | Code |
|---|---|
| Base Cabinet 1 Door, Right | `BC1D_R` |
| Base Cabinet 1 Door, Left | `BC1D_L` |
| Base Cabinet 2 Door | `BC2D` |
| Base Cabinet 2 Door w/ Divider | `BC2D_WD` |
| Base Cabinet 1 Door 1 Drawer, Right | `BC1D1D_R` |
| Base Cabinet 1 Door 1 Drawer, Left | `BC1D1D_L` |
| Base Cabinet 2 Door 1 Drawer | `BC2D1D` |
| Base Cabinet 2 Door 2 Drawer | `BC2D2D` |
| Base Cabinet 2 Door 2 Drawer w/ Divider | `BC2D2D_WD` |
| Base Cabinet 3 Drawer | `BDS3` |
| Base Cabinet 4 Drawer | `BDS4` |
| Base Open Cabinet | `BCO` |
| Base Sink Cabinet 2 Door | `BSC2D` |
| Base Sink Cabinet 2 Door, ADA | `BS2D_ADA` |
| Base Sink Cabinet 2 Door, 1 False Front, ADA | `BS_ADA` |
| Base Trash Cabinet 1 Drawer | `BCT1D` |
| Base Trash Cabinet 2 Drawer | `BCT2D` |
| Base Trash Cabinet Pullout | `BCTPO` |
| Base Trash Cabinet with Bottom, 1 Door, Left | `BCT1DL` |
| Base Trash Cabinet with Bottom, 1 Door, Right | `BCT1DR` |
| Base Trash Cabinet No Bottom, 1 Door, Left | `BCTNB1DL` |
| Base Trash Cabinet No Bottom, 1 Door, Right | `BCTNB1DR` |
| Base Trash Cabinet No Bottom, 2 Door | `BCTNB2D` |
| Pencil Drawer 1 Drawer | `PD1D` |
| Pencil Drawer 2 Drawer | `PD2D` |
| Ladder Base | `LB` |

> **Note:** [Base Cabinets](/reference/construction-standards/casework/base-cabinets.html#type-specific-notes) refers to the pullout trash cabinet as `BCT_PO` (with an underscore); this list uses `BCTPO`. Confirm which exact string is authoritative in Epicor before using either in shop documentation.

Construction detail for base cabinet types is documented in [Base Cabinets](/reference/construction-standards/casework/base-cabinets.html); drawer-specific rules (including pencil drawers) are in [Drawer Construction](/reference/construction-standards/casework/drawer-construction.html).

## Upper Cabinets

| Name | Code |
|---|---|
| Upper Cabinet 1 Door, Right | `UC1D_R` |
| Upper Cabinet 1 Door, Left | `UC1D_L` |
| Upper Cabinet 2 Door | `UC2D` |
| Upper Cabinet 2 Door, No Divider | `UC2D_ND` |
| Upper Cabinet Open | `UCO` |

> **Note:** The drawing set still labels this cabinet `UC2D_WD` ("with divider"). The current, correct code is `UC2D_ND` — **no divider**. Don't rely on the drawing's own label or description for this one.

Construction detail for upper cabinets is documented in [Upper Cabinets](/reference/construction-standards/casework/upper-cabinets.html).

## Fillers

| Name | Code |
|---|---|
| Base Filler | `BF` |
| Base Corner Filler | `BCF` |
| Base Filler ADA, Right | `BF_ADA_R` |
| Base Filler ADA, Left | `BF_ADA_L` |
| Upper Filler, Right | `UF_R` |
| Upper Filler, Left | `UF_L` |
| Upper Corner Filler | `UCF` |
| Tall Filler | `TF` |

Construction detail for base and upper fillers is documented in [Fillers](/reference/construction-standards/casework/fillers.html). Tall Filler (`TF`) is named here but does not yet have documented construction rules — confirm with engineering before fabrication if you need this type.
