# Remote Desktop (Splashtop)

A shared remote desktop machine is available to the engineering team via Splashtop. The machine is hosted at the Colorado facility and provides access to heavy-use software that is needed infrequently and is not practical to install on individual workstations.

## Available Software

- **Revit** — BIM authoring and coordination
- **Navisworks** — Model aggregation, clash detection, and BIM review
- **Leica BLK360 (Cyclone)** — Laser scan processing software for point cloud registration and export
- Additional software as needed

## Access

The machine is accessed via **Splashtop** using shared engineering team credentials. Login details are distributed through internal team channels — contact your lead or department management if you need access.

## Usage Notes

- This is a **shared machine** — coordinate with teammates before starting a long session to avoid conflicts
- Do not save project files locally on the remote machine; work from Box or transfer files to your local machine when done
- Log out of Splashtop when finished to free the session for others
